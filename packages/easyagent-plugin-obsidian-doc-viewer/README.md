# EasyAgent Plugin: Obsidian Doc Viewer

> 将 Doc Project 的 Obsidian-like Markdown 浏览器打包为 EasyAgent 插件

## 功能

- 在 EasyAgent 中打开项目文档目录
- 左侧目录树浏览
- 中间关系图谱展示 WikiLink 网络
- 右侧 Markdown 预览
- 全文搜索

## 安装

1. 在 EasyAgent 设置 → 第三方插件 → 社区插件市场中搜索 "Obsidian Doc Viewer"
2. 点击安装并启用

## 使用

安装后，通过命令面板或侧边栏入口打开 "Doc Viewer"。
