## 二、详细实施方案

### 🔴 P0 阶段：紧急修复（今日，合计 ~55min）

#### I01 — 修复 `frontend/package.json` exports

```diff
// packages/frontend/package.json
- "main": "./src/main.tsx",
+ "main": "./src/mountApp.tsx",
  "exports": {
-   ".": "./src/main.tsx",
+   ".": "./src/mountApp.tsx",
```

**验证**：`node -e "require('@easyagent/frontend')"` 不再报 MODULE_NOT_FOUND。

#### I02 — 修复 ESLint 配置

```bash
pnpm add -D -w eslint @eslint/js typescript-eslint eslint-plugin-react-hooks
```

创建 `eslint.config.mjs`（根目录，Flat Config 格式）：

```js
import js from '@eslint/js';
import tseslint from 'typescript-eslint';
import reactHooks from 'eslint-plugin-react-hooks';

export default tseslint.config(
  js.configs.recommended,
  ...tseslint.configs.recommended,
  {
    plugins: { 'react-hooks': reactHooks },
    rules: {
      'no-console': 'warn',
      '@typescript-eslint/no-explicit-any': 'warn',
      'react-hooks/rules-of-hooks': 'error',
      'react-hooks/exhaustive-deps': 'warn',
    },
  },
  { ignores: ['dist/**', 'node_modules/**', 'release/**', '*.config.*'] },
);
```

更新根 `package.json`：

```diff
  "lint": "eslint packages/*/src --ext .ts,.tsx --max-warnings 50",
+ "lint": "eslint packages/*/src --max-warnings 50",
  "lint:fix": "eslint packages/*/src --fix",
```

#### I03 — 跨平台兼容性

```bash
pnpm add -D -w cross-env rimraf
```

```diff
// 根 package.json
- "clean": "rm -rf packages/*/dist packages/*/node_modules/.cache",
+ "clean": "rimraf packages/*/dist packages/*/node_modules/.cache",

// packages/web/package.json
- "dev": "set NODE_OPTIONS=--no-deprecation && vite",
+ "dev": "cross-env NODE_OPTIONS=--no-deprecation vite",
```
