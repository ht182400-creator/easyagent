# Memory 目录

本目录存放项目的跨会话记忆文件。

## 文件说明

| 文件 | 用途 |
|------|------|
| `YYYY-MM-DD.md` | 每日工作日志（自动生成，追加式） |
| `MEMORY.md` | 长期记忆/项目约定（手动维护） |

## 格式规范

编写 memory 文件时，请遵循以下规范，以便管线解析器正确提取问题数据：

> 完整规范 → `docs/pipeline/memory-format-spec.md`

### 核心要求

1. 需要被管线页面索引的问题使用 `## [模块:ID] 标题` 格式
2. 必须包含 `- **问题**:` / `- **修复**:` / `- **状态**:` 字段
3. 纯操作流程（启动/构建/发布）不加 `[模块:ID]` 标签
4. 状态用 emoji：`✅ resolved` / `⏳ pending` / `❌ open`
